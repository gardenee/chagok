# 설정 탭 추가 & 탭 구조 개편 설계

> 작성일: 2026-03-08

## 배경

현재 탭 구조(홈 · 캘린더 · 예산·결산 · 자산 · 고정지출)에서 홈 탭을 제거하고 설정 탭을 추가한다.
홈 대시보드는 MVP 이후 별도 검토.

---

## 탭 구조 변경

### AS-IS (5개)
홈 · 캘린더 · 예산·결산 · 자산 · 고정지출

### TO-BE (5개)
고정지출 · 예산 · 캘린더 · 자산 · 설정

| 순서 | 탭명 | 파일 | 아이콘 | 변경사항 |
|------|------|------|--------|---------|
| 1 | 고정지출 | `fixed.tsx` | Repeat | 순서 이동 |
| 2 | 예산 | `budget.tsx` | BookCheck | 이름 변경 (예산·결산 → 예산) |
| 3 | 캘린더 | `calendar.tsx` | CalendarDays | 순서 이동 |
| 4 | 자산 | `assets.tsx` | Landmark | 순서 유지 |
| 5 | 설정 | `settings.tsx` | Settings | 신규 추가 |

- `app/(tabs)/index.tsx` (홈) 제거
- `app/(tabs)/settings.tsx` 신규 생성

## 설정 화면 설계

### 레이아웃
- 배경: `cream`
- 상단: 타이틀 "설정" (Typography.title)
- 섹션별 그루핑된 리스트 형태
- 섹션 헤더: brown/50 색상, caption 크기

### 섹션 구성

#### 우리 커플
| 항목 | 동작 |
|------|------|
| 가계부 이름 | 탭 → 편집 모달 또는 인라인 편집 |
| 초대코드 | 탭 → 코드 표시 + 공유하기 버튼 |
| 파트너 | 파트너 닉네임 표시 (읽기 전용) |

#### 내 정보
| 항목 | 동작 |
|------|------|
| 닉네임 | 탭 → 편집 모달 또는 인라인 편집 |

#### 알림 *(UI만 구현, 실제 동작은 MVP 이후)*
| 항목 | 동작 |
|------|------|
| 파트너 지출 알림 | Toggle (Switch) |
| 댓글 알림 | Toggle (Switch) |
| 고정지출 리마인더 | Toggle (Switch) |

#### 계정
| 항목 | 동작 | 스타일 |
|------|------|--------|
| 로그아웃 | 탭 → 확인 다이얼로그 → Supabase signOut | 기본 |
| 계정 탈퇴 | 탭 → 확인 다이얼로그 → 데이터 삭제 | 빨간색 (위험 영역) |

## 구현 범위

### 포함
- 탭 레이아웃 개편 (`_layout.tsx` 수정)
- 홈 탭 제거 (파일 삭제)
- 설정 화면 UI 구현
- 가계부 이름 편집 (Supabase `couples` 테이블 update)
- 닉네임 편집 (Supabase `users` 테이블 update)
- 초대코드 표시 + 공유하기 (Share API)
- 파트너 정보 표시 (읽기 전용)
- 로그아웃
- 알림 Toggle UI (비활성 상태로 표시)

### 제외 (MVP 이후)
- 알림 실제 동작 (FCM 연동)
- 계정 탈퇴 실제 처리
- 프로필 사진
